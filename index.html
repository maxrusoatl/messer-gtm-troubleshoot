Signal Health

  • Endpoint mismatch prevents /data events from reaching sGTM.
  • Data Tag fires only on purchase, missing key ecommerce events.
  • User-ID mapping is hashed and stored in the wrong path.

Compliance Risk

  • Raw PII can be sent via add_data_layer without hashing.
  • Consent signals are uneven across wGTM and sGTM tags.
  • Event IDs differ between client and server flows.

Fast Wins

  • Align gtm_server_domain to the actual sGTM endpoint.
  • Trigger Data Tag on all ecommerce events.
  • Add GA4 tags for view_item, view_cart, begin_checkout.
