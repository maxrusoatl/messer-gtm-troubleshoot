Root Cause: Data Tag Trigger Misconfiguration

The wGTM Data Tag only fires on purchase events. All other ecommerce events are not forwarded to sGTM.

Single Fix Required

Update the Data Tag trigger in GTM-MNRP4PF to fire on ALL ecommerce events.

  1. Open GTM-MNRP4PF in Google Tag Manager
  2. Navigate to Tags → Data Tag
  3. Remove trigger ce_purchase (ID 96)
  4. Add trigger Trigger - Ecom Core Events (ID 313)
  5. Save and Publish
